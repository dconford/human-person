Common Terms

  • HTTP: HyperText Transfer Protocol
  • HTML: HyperText Markup Language
  • TCP: Transmission Control Protocol
  • URL: Uniform Resource Locator

The hypertext transfer protocol allows information to be transferred over a network. When a transfer takes place, there is a serer and a client involved. The client, usually through a browser, sends a request for information. The server then responds or serves the requesed information if the request was valid. If your request was not valid you may get something like this: 404 The transaction usually takes place over a TCP connection.

Block Level vs In-Line

Block elements start on a new line and may contain in-line elements nested within them. In-line elements do not start on a new line by default, they stay in the flow of the text.

Semantic HTML & Accessibility

Semantic HTML is the use of HTML markup to reinforce the semantics, or meaning, of the information in webpages and web applications rather than merely to define its presentation or look. Semantic HTML is processed by traditional web browsers as well as by many other user agents. CSS is used to suggest its presentation to human users.
